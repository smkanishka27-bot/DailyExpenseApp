App Discription

The Daily Expense App is a simple financial tracking tool that helps users manage daily spending efficiently. It allows users to enter expense details such as amount, 
description, category, and payment mode, and displays records in a structured ListView layout. The application also provides quick communication options to call, send SMS, 
or email the accountant using Intents after confirmation through a DialogBox. It demonstrates practical implementation of Android UI components, data handling, and real-time 
interaction features.
